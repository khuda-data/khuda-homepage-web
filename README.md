# KHUDA Homepage

경희대학교 데이터/AI 학회 KHUDA 공식 홈페이지

## 실행

```sh
npm i
npm run dev
```

## 관리자 페이지

질문 목록 테스트 및 향후 설정 관리를 위한 관리자 페이지가 있습니다.

### 접근 방법

1. 브라우저에서 `/admin` 경로로 접근
2. 비밀번호 입력 (환경변수로 설정 가능)
3. 환경변수 설정: `.env` 파일에 `VITE_ADMIN_PASSWORD=비밀번호` 추가

### 기능

- **API 테스트**: 질문 목록 API 호출 테스트
- **설정 관리** (준비 중): 다음 기수에서 질문 및 옵션 관리 기능 추가 예정
